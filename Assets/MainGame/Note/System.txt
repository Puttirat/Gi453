Character System
Character Movement - WASD.Movement, Ctrl.Crouch, No Jump, เดินขึ้นเนินได้
Pick Up Item | Flashlight - หยิบแล้วนำมาถือไว้ที่มือข้างซ้าย ไม่สามารถมารถโยนทิ้งได้
Pick Up Item - หยิบแล้วนำมาถือไว้มือข้างขวา สามารถโยนได้
Pick Up Item | Muti item - Item ที่มีขนาดใหญ่และสามารถถือหลายอันได้
Core item Pick Up - Floppy disk หรือสิ่งของ Quest ที่มีขนาดเล็ก เมื่อเก็บแล้วจะถูกเก็บไว้ในตัวละครไม่แสดงให้ผู้เล่นเห็น
Disk System - เมื่อเก็บ Item นี่แล้วจะถูกเก็บไว้ตัวผู้เล่น เมื่อนำไปใส่ที่เครื่องใส่ Disk หากมีมากกว่า 1 อันจะต้องแสดง UI ให้ผู้เล่นเลือก Disk ที่จะใส่

Light System
Flsh Light System - เปิดปิดได้ ยิงแสงในรูปแบบกรวย
Universal Light - ระบบแสงภายในเกมสามารถปรับเช้ากลางวันได้

Evnet Trigger System
Type 1 - ผู้เล่นเดินถึงจุดที่กำหนด Event จะทำงาน
Type 2 - ระยะเวลาถึง เวลาที่กำหนด Event จะทำงาน
Type 3 - เมื่อผู้เล่นหบิบสิ่งของที่กำหนด Event จะทำงาน
Type 4 - ยิงตัวเช็คออกจากด้านหน้าผู้เล่นเป็นเส้นตรงหากโดนกับสิ่งของที่กำหนด Event จะทำงาน

Audio System
Dynamic Sound - ระบบเสียงแบบ 3D มีใกล้ไกล ทำให้สามารถปรับระยะของเสียงได้

Save System
Auto Save/Load - ระบบ Auto Save และโหลด Save ไปจุดที่ Auto Save ไว้
Respawn - เมื่อตายผู้เล่นตายจอคอมตะดับและเมื่อผู้เล่นเข้าไปในคอมอีกรอบตัวเกมจะกลับไปจุด Auto Save ล่าสุด
Save Delete - เมื่อจบด่านตัวเกมจะลบ Save เพื่อไม่ให้ผู้เล่นย้อยกลับไป Auto Save ล่าสุดได้

Change Room
- เปลี่ยนระหว่างโลกในเทปกับในห้องผ่านจอคอมพิวเตอร์เมื่ออยู่ฝั่งห้องตัวเกมในเทปจะหยุด

Other System
*Transition - เวลาเปลี่ยนฉากทำให้จอ Fade out เปลี่ยนฉากและจึง Fade in 
Loading System - ทำระบบรอโหลด Asset ก่อนเข้าด่าน
*Disk Cmd - เมื่อผู้เล่นใส่ Disk เข้าไปในคอมแล้วจะสามารถกดปุ่ม "?" เพื่อเปิด Cmd ได้และสามารถพิมคำสั่งเพื่อสร้างสิ่งของจาก Keyword ใกล้เคียงได้
*Cutscene - ตัวเข้า cutscene แล้วในระหว่างนั้นอาจจะให้ตัวเกมทำการโหลด Asset ไปด้วย
